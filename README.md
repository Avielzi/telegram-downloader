##  English

**Telegram Downloader** is a modern, professional tool designed for bulk media downloading from Telegram channels and groups. Built with PyQt6 and Telethon, it offers a stable and user-friendly experience.

###  Features
- **Smart Bulk Download**: Download hundreds of files with a single click.
- **Automatic Scanning**: Find all media within a group/channel easily.
- **Visual Selection**: Choose exactly what you want to download using a clear table view.
- **Advanced Filtering**: Filter by Photos, Videos, Documents, and Archives.
- **Modern UI**: Material Design interface with real-time progress bars.
- **Multi-language Support**: Fully supports English, Hebrew, Spanish, Russian, and Arabic.
- **Secure**: Connects directly to the official Telegram API; all data stays on your machine.

###  Installation
1. **Install Python**: Ensure Python 3.8+ is installed (Check "Add Python to PATH").
2. **Install Dependencies**:
   - Double-click `install.bat` (Recommended)
   - Or run: `pip install -r requirements.txt`
3. **Run**: Double-click `run.bat`.

###  Quick Start
1. **Setup**: Enter your API ID, API Hash, and Phone Number (Get them from [my.telegram.org](https://my.telegram.org)).
2. **Scan**: Paste a group/channel link and start scanning.
3. **Select**: Use the filters and checkboxes to select files.
4. **Download**: Choose a destination folder and start downloading!
